# soluxTimeTable (2020.12.21 ~ )
* 인프런 - 안드로이드 스튜디오로 안드로이드 앱 만들기(수강신청 도우미 어플리케이션 개발부터 마켓 출시까지)                       
                    
                    
                                 
### 개발환경 및 사용언어
1. Android Studio(Java)
2. Cafe24                  
                    
                    
                         
### 구현 내용
* 로그인, 회원가입(비밀번호 암호화, 중복체크)
* 공지사항, 강의목록, 강의삭제
* 시간표(중복제한, 학점제한)
* 강의분석(학과, 성별등에 따른 인기 순위) 
* 어플리케이션 정보(팝업창)
* 일정정리 - 구현중
* 우선순위 - 구현중
